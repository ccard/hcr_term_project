Rocon
=====

This is a meta-repository for the `Robotics in Concert`_ organisation
on github which gets used to house rosinstallers, provide a meta-package
and handle organisation-wide issues and discussions. Here you can find
general guidelines relevant to developing across all of
the rocon repositories.
