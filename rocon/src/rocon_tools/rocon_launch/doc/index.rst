Rocon Launch
============

Extends roslaunch to be a multi-roslaunch utility - i.e. can spawn multiple terminals inside each of which
executes a single roslaunch (single or multi-master variety). This is used extensively in the `Robotics in Concert`_
framework for testing and with simulations.

.. _`Robotics in Concert`: http://www.robotconcert.org/wiki/Main_Page
