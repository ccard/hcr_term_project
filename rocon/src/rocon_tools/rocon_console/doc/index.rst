Rocon Console
=============

This package provides a few useful console utilities for the pythonic parts
of the `Robotics in Concert`_ framework. Right now these are primarily for
colourising output on the console for ease of conveying information.

.. _`Robotics in Concert`: http://www.robotconcert.org/wiki/Main_Page
